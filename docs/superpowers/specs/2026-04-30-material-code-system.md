# 物料编码系统设计

**日期**: 2026-04-30
**版本**: v1.0

---

## 1. 编码结构

采用 **18位阿拉伯数字编码**，分为8个层级：

```
XX    XX    XX    XX    XX    XX    XX    XXXX
大类   中类   小类   规格   单位   供应商  年份   流水号
(2位) (2位) (2位) (2位) (2位) (2位) (2位)  (4位)
```

### 1.1 大类编码 (第1-2位)

| 编码 | 类别 | 说明 |
|------|------|------|
| 01 | 钢材类 | 钢筋、钢板、钢管、钢构件 |
| 02 | 水泥类 | 普通水泥、特种水泥 |
| 03 | 混凝土类 | 现场搅拌混凝土、预制混凝土 |
| 04 | 电缆类 | 电力电缆、控制电缆、通信电缆 |
| 05 | 管材类 | PVC管、钢管、PE管 |
| 06 | 木材类 | 模板、木方、竹胶板 |
| 07 | 电气类 | 开关、插座、灯具、配电箱 |
| 08 | 设备类 | 机械设备、电动工具、测量仪器 |
| 09 | 五金类 | 螺丝、螺母、铁丝、钢丝绳 |
| 10 | 防护类 | 安全帽、安全带、防护网 |
| 11 | 油漆涂料类 | 油漆、涂料、稀释剂 |
| 12 | 保温防水类 | 保温材料、防水卷材、密封胶 |
| 99 | 其他 | 不属于上述类的材料 |

### 1.4 规格编码 (第7-8位)

表示规格型号，如：
- 01 = 标准规格
- 02 = 大规格
- 03 = 小规格

具体规格可在系统中定义。

### 1.5 单位编码 (第9-10位)

| 编码 | 单位 | 说明 |
|------|------|------|
| 01 | 吨 | t |
| 02 | 千克 | kg |
| 03 | 米 | m |
| 04 | 根 | 根 |
| 05 | 个 | 个 |
| 06 | 套 | 套 |
| 07 | 卷 | 卷 |
| 08 | 箱 | 箱 |
| 09 | 块 | 块 |
| 10 | 平方 | m² |
| 11 | 立方 | m³ |
| 12 | 包 | 包 |

### 1.6 供应商编码 (第11-12位)

两位数字，代表供应商编号（00-99），可在系统中维护供应商列表。

### 1.7 年份编码 (第13-14位)

两位数字，表示入库年份末两位：
- 26 = 2026年
- 27 = 2027年

### 1.8 流水号 (第15-18位)

四位数字，0001-9999，同一年份同一小类内顺序递增。

## 3. 编码生成规则

### 3.1 首次入库
1. 用户输入物料名称
2. 系统根据物料名称匹配大类、中类、小类
3. 如果没有匹配的分类，提示用户选择或创建新分类
4. 系统自动分配流水号

### 3.2 已有物料入库
1. 用户输入物料名称或编码
2. 系统自动带出已有的编码信息
3. 用户确认后入库

### 3.3 分类扩展
- 当遇到新材料时，可扩展中类和小类编码
- 扩展规则：大类固定，中类和小类按顺序添加
- 确保编码的唯一性

## 5. 系统功能

### 5.1 自动编码
- 新物料入库时自动生成编码
- 根据物料名称智能匹配分类
- 如果无法匹配，提示用户手动选择分类

### 5.2 编码查询
- 支持按编码查询
- 支持按物料名称模糊查询
- 支持按分类层级查询

### 5.3 编码维护
- 管理员可维护分类体系
- 可添加新的中类和小类
- 可修改分类名称和说明
