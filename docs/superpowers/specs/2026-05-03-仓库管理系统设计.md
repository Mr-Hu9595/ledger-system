# 仓库管理系统设计方案

> **状态**：设计完成，等待实现

## 1. 项目概述

**项目名称**：仓库管理系统

**项目类型**：Electron 桌面应用 + React 前端 + Python FastAPI 后端

**核心功能**：
- 物料搜索查询（模糊匹配）
- 入库/出库记录管理
- 库存查看与预警
- Excel 批量导入
- 数据导出（Excel/PDF）

**目标用户**：单人使用的建筑工地材料管理员

**技术架构**：
```
┌─────────────────────────────────────────────────────────┐
│  Electron 28 桌面应用窗口                                │
│  ┌───────────────────────────────────────────────────┐  │
│  │  React 18 + Ant Design Pro + Vite               │  │
│  │  ─────────────────────────────────────────────── │  │
│  │  页面：首页看板 | 物料管理 | 入库管理 | 出库管理  │  │
│  │       | 工具（导出/同步/编码）                     │  │
│  └───────────────────────────────────────────────────┘  │
│                         ↓ HTTP (localhost:8000)        │
│  ┌───────────────────────────────────────────────────┐  │
│  │  Python FastAPI 后端服务                          │  │
│  │  ─────────────────────────────────────────────── │  │
│  │  API: /api/materials | /api/inbound | /api/outbound │
│  │       /api/import | /api/export                    │  │
│  └───────────────────────────────────────────────────┘  │
│                         ↓ SQLAlchemy                    │
│  ┌───────────────────────────────────────────────────┐  │
│  │  PostgreSQL 数据库                                │  │
│  │  表：ledger, ledger_property, inbound, outbound    │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

- `inbound` - 入库记录表
- `outbound` - 出库记录表

## 4. API 设计

### 4.1 物料接口

```
GET    /api/materials              # 获取物料列表（支持分页、搜索、筛选）
GET    /api/materials/:id          # 获取物料详情
POST   /api/materials              # 新增物料
PUT    /api/materials/:id          # 更新物料
DELETE /api/materials/:id          # 删除物料
```

### 4.2 入库接口

```
GET    /api/inbounds               # 获取入库记录
POST   /api/inbounds               # 新增入库
GET    /api/inbounds/:id           # 获取入库详情
```

### 4.3 出库接口

```
GET    /api/outbounds              # 获取出库记录
POST   /api/outbounds              # 新增出库
GET    /api/outbounds/:id          # 获取出库详情
```

### 4.4 工具接口

```
POST   /api/import/excel           # 导入Excel
GET    /api/export/excel           # 导出Excel
POST   /api/sync/report            # 同步报表
```
